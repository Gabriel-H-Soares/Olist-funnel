# Olist Funnel Analysis

This repository contains a portfolio project focused on analyzing the Olist sales funnel.

## What's inside

- `Project_olis_funnel.ipynb`: exploratory and business-oriented data analysis in Jupyter.
- `streamlit_olist_funnel.py`: interactive Streamlit app to explore the funnel metrics and insights.
